# Klaytn v1.4.0 릴리즈 정보 <a id="release-notes-for-klaytn-v1-4-0"></a>

## New Features <a id="new-features"></a>
- 키스토어 V4
  - Klaytn 계정 활용을 돕는 새로운 키스토어 표준  [KIP-3](https://klaytn.github.io/kips/KIPs/kip-3)을 도입했습니다.

## Fixes <a id='fixes'></a>
- `stakingInfo` 업데이트 시 타이밍 이슈로 인해 간간이 잘못된 블록이 발생하는 문제를 해결했습니다.
- API와 트리 노드 가비지 컬렉션 사이의 타이밍 이슈로 인해 `debug.traceBlock` 실행 시 발생하는 문제를 해결했습니다.
- 블록 동기화 과정 중 보고된 버그를 수정했습니다.

## Improvements <a id='improvements'></a>
- 최적화된 스토리지 트리
  - Klaytn은 이제 모든 트랜잭션이 아니라 모든 블록마다 스토리지 루트 해시를 계산합니다.
  - Klaytn은 트리 해시 루트를 동시에 계산합니다.
- 증가된 상태 트리 캐시 크기
  - 기본 캐시 크기 증가로 향상된 성능
- Badger DB 버전 v1.6.0으로 업그레이드.
- 매트릭스 패키지 교체
- 업데이트된 패키지 설명
- 더 넓어진 테스트 코드 커버리지
