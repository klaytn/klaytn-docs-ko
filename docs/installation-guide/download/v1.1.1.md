- 더 많은 테스트 케이스 추가
- 업데이트된 패키지 설명

# Klaytn v1.1.0 릴리즈 정보 <a id="release-notes-for-klaytn-v1-1-0"></a>

### 새로운 기능 <a id="new-features"></a>
- 특정 블록 번호에서 스마트 컨트랙트 메소드 호출 지원
- 거버넌스 의회 멤버를 위한 여러 스테이킹(staking) 컨트랙트 지원

### Fixes <a id="fixes"></a>
- Fixed various bugs

### Improvements <a id="improvements"></a>
- 컨센서스 메시지는 거버넌스 의회 멤버에게 전파되어 라운드 번호를 동기화
- 리팩토링된 패키지(보상, 거버넌스 등)
- 업데이트된 doc.go의 패키지 설명
- Added more unit tests
